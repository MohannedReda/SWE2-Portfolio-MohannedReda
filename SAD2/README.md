# System Analysis and Design 2 (SAD2)

This folder contains all tasks and assignments related to the **System Analysis and Design 2** course.

## Course Overview

The course focuses on analyzing system requirements and designing system models using standard analysis and design techniques such as UML diagrams and documentation.

## Folder Structure

- **Task01_taskname/**  
  Contains the deliverables of Task 01, including reports and related diagrams.

- **Task02_taskname/**  
  Contains the deliverables of Task 02.

- **Task03_Class_taskname/**  
  Contains class-based tasks such as Class Diagrams and related documentation.

- **Task0N_Class_taskname/**  
  Contains advanced or final class-related tasks.

Each task folder includes:
- A README.md explaining the task
- Required documents, diagrams, or reports

## Tools Used

- UML Design Tools (e.g., StarUML, draw.io)
- Documentation tools (PDF / Word)

## Notes

- Tasks are organized sequentially and clearly named.
- Each task folder contains all files required for evaluation.
- This folder is prepared strictly for academic submission.
